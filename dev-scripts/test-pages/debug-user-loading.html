This page will open the main app and check the console for user loading states.

Instructions:

  1. Click "Open Main App" below
  2. Open Developer Tools (F12) in the new tab
  3. Look for user-related console logs
  4. Check if currentUser is null/undefined vs an actual user object

What to look for:

  • User loading logs: Look for [UserManagement] or [GuestUsers] logs
  • Theme provider logs: Look for [UserTheme] logs
  • Current user value: Should show actual user object, not undefined
  • Emergency disable status: Should be false

Console Debug Commands:

Copy these into the browser console:

// Check localStorage emergency flags
console.log('Emergency Disabled:', localStorage.getItem('user_theme_isolation_disabled'));

// Check if user context is working
console.log('Window location:', window.location.href);

// Check environment variable
console.log('Environment variable:', process.env.NEXT_PUBLIC_ENABLE_USER_THEME_ISOLATION);

// Look for React DevTools to inspect component state
console.log('React DevTools available:', typeof window.__REACT_DEVTOOLS_GLOBAL_HOOK__ !== 'undefined');
